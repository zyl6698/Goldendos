# Goldendos

Goldendos is a HTTP DoS Test Tool.  
Fork from [GoldenEye](https://github.com/jseidl/GoldenEye.git). 
This Shell is made for Centos 7+,enjoy it !  

##Usage##

```
git clone https://github.com/zyl6698/Goldendos.git &&cd Goldendos
chmod +x hack1.0.sh
./hack1.0.sh
```
